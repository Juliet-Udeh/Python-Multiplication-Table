# Dynamic Multiplication Table Generator

A lightweight, interactive Python script that generates multiplication tables based on user-defined limits. This project demonstrates clean use of loops, user input handling and arithmetic logic in Python.

## Features
* **Customizable Base:** Choose any number to multiply.
* **Customizable Range:** Define how far the table should go (e.g., up to 10, 12, or 100).
* **Loop Logic:** Utilizes the `for` loop and `range()` function for efficient execution.

## How to Run
Ensure you have [Python](https://www.python.org/) installed.
